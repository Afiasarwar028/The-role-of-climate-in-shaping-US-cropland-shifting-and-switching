Crop data
Harvested .area data are obtained from the HarvestGRID dataset. Data are provided at the county level (GEOID) and span the period 1981–2019.

Climate data
Climate variables are derived from the PRISM dataset and are aggregated to the county level to match the spatial resolution of the crop data.

Spatial and temporal units

Spatial unit: U.S. counties (GEOID)

Temporal coverage: 1981–2019 (annual values)

Data processing
All filtering, detrending, standardization, and aggregation steps are performed using the R scripts in the scripts/ directory.
No processed datasets are stored in this repository
